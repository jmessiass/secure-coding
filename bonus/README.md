## Como Usar

1. **Salve ou bookmark** estes materiais
2. **No code review**, abra o checklist (#02) e valide cada item
3. **Antes de commitar**, passe o prompt de revisão geral (#03, Prompt 1) no seu código
4. **Na dúvida**, consulte o guia de bolso (#01) pela categoria OWASP
5. **Compartilhe** com colegas que não participaram do treinamento
